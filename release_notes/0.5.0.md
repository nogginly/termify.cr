# `termify.cr`

## Release 0.5.0

### FEATURES

- *Markdown → Terminal renderer* – Render standard Markdown directly to a terminal, streaming output when compiled with `-Dstreaming_highlight`.
- *Syntax‑highlighters* – Built‑in Tartrazine support for code fences; block‑style and inline‑style code with line‑numbering, gutter styles, and theme selection.
- *Tables* – Full table rendering via Tablo; automatic width, alignment, and packed layout handling.
- *Styling* – Declarative `Termify.markdown_stylesheet` allows per‑element styles (`bold`, `fg`, `bg`, `dim`, `line_prefix`, `newline_before/after`, `line_number_format`, `highlight_theme`, `gutter_style`, etc.).
- *Color handling* – ANSI colors via `ANSI::Color`, support for hex and named `Color256` enums, merged stylesheet support.
- *Block & list support* – Nested lists, blockquotes (prefix “│ ”), block continuations, and proper whitespace/newline management.
- *API* – `Termify.render_markdown([io],[stylesheet]){ |md_io| … }` streams Markdown; `Termify.markdown_stylesheet` returns a ready‑to‑use stylesheet.
- *Tests & CI* – Full test coverage for syntax highlighting, color parsing, stylesheet merging, table rendering, and markdown edge cases.
