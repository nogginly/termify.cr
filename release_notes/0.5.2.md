# `termify.cr`

## Release 0.5.2

### IMPROVEMENTS

- Add `Renderer#reset` method that resets the renderm, closing off any blocks and returning it to a fresh start. Useful when the caller doesn't want to close the renderer but knows a reset is warranted.

### FIXES

- Remove unnecessary attempt to flush lines for every write unless chunk has newline. (Optimization)
- Fixed a degenerate table buffering case where single column divider row doesn't match when surrounding bars are removed
